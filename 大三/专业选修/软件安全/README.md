### COMP501027 软件安全分析

授课教师：范铭 [Welcome - 范 铭 - 教师个人主页](https://gr.xjtu.edu.cn/en/web/mingfan)

考察方式：考核，三次小实验 + pre + 项目 + 项目报告。三次小实验给的教程特别详细（几乎是手把手教），课上老师会先演示一遍，实验报告也是简略写一下就行，课上可能会找同学进行复现；后三者都是围绕项目开展的，老师会给参考选题，不会要求做得很深很完备，主要还是探索兴趣，不用害怕太难不会做。

老师挺好的，会叫人回答问题但是答不上来也没事，给分总体都很不错。内容涉及一些逆向的知识，有一定实用性，最好有一些汇编知识的基础。实验挺有趣的，也不难，新手可上手的一些逆向工程，做出来会很有成就感。
