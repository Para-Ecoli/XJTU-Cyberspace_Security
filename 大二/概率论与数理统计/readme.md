### 概率论与数理统计

我们是电信学部接手这门课的第一年，此前都是数学学院开课的，讲课内容和考试内容都有点差别，找复习资料的时候可以参考，但要了解两边出题风格是不太一样的。

第一次开课最后两章（随机过程）甚至没讲完，最后考试也没考，但是这部分实际上在cs的应用里是很重要的，有条件还是要学一下。

我个人基本上没咋听过课（感觉让计算机的老师来讲数学原理的课程还是多少有点为难他们了），纯靠b站网课自学最后考了85+，也有可能是第一年出题没经验的原因，期末考试题有点怪怪的，有一道大题是古早考研原题（很简单），还连着考了两题条件概率公式（完全一样的知识点连着考了两个大题）。概率论和数理统计部分可以按照考研的内容去学习，不知道近年教学风格和考试内容上有没有大的变动。

这里的复习PPT是数学学院的这门课的相关材料，基础内容是普适的，可以参考复习。笔记部分是我听考研网课自学结合课内强调的一些点写的，基本能涵盖基础知识，作为复习的参考，大家还是以老师课内讲的为准。
